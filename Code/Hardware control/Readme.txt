Readme for code to control Athesim hardware
******************************************************

The following code is provided to show the method(s) of hardware control
and can be used as a starting point for developing future systems.

The main Labview vi, MAI-SIM control.vi, is a GUI for control of all hardware.
take image.vi is a dependency and forms the basis of the mirror/camera control 
process. MAI-SIM control.vi in the development branch is continually updated
to extend the system capabilities. 

Hardware connections.jpg shows the harware connections used in the system.
